# **App Name**: DocuFlow

## Core Features:

- Secure PDF Upload: Users can securely upload PDF files into the system, supporting multiple file uploads.
- Intuitive Folder Management: Create, rename, organize, and delete folders to maintain a structured hierarchical organization for all documents.
- Integrated PDF Viewer: View uploaded PDF documents directly within the application, eliminating the need for external software.
- Document Browsing & Navigation: Effortlessly navigate through folders and documents via a user-friendly interface.
- AI-Powered Document Tagging Tool: An AI tool will analyze uploaded PDF content to suggest relevant tags and keywords automatically, enhancing organization and discoverability.
- Smart Document Search: Search for documents by filename, folder name, and leverage AI-suggested tags for content-aware searching.
- Responsive User Interface: A clean, consistent, and responsive interface optimized for efficient document management across devices.

## Style Guidelines:

- Primary color: Professional and clear blue (#2E5FA9), embodying reliability and intelligence.
- Background color: A subtle, light blue-grey (#F1F3F5), providing a clean and non-distracting canvas.
- Accent color: A vibrant, clear cyan (#31C0ED), used for interactive elements and highlights to ensure visibility and engagement.
- Headlines font: 'Space Grotesk', a proportional sans-serif for a modern and tech-informed feel. Body font: 'Inter', a neutral grotesque-style sans-serif for excellent readability in longer text blocks.
- Use crisp, minimalist outline icons for file operations, navigation, and system functions to maintain a modern and uncluttered aesthetic.
- A classic two-column ERP-like layout, with a fixed sidebar for folder navigation and a main content area for document listing and viewing, ensuring intuitive user flow.
- Implement subtle, fast transitions for folder expansions, document loading, and successful upload notifications to enhance user feedback without being intrusive.
